# 🔐 Simple Login UI

A clean and minimal login page built using **HTML and CSS** — no JavaScript, no frameworks. Created while practicing frontend basics and improving UI design skills.

## ✨ Features

- 🌄 Full-screen background image with dark overlay
- 🃏 Semi-transparent dark card layout
- 📧 Email input field with mail icon
- 🔒 Password input field with lock icon
- ☑ Remember me checkbox
- 🔘 Pill-shaped Login button
- Clean, minimal dark theme

---

## 🛠 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and form elements |
| CSS3 | Styling, layout, background, transparency |

> Pure HTML & CSS — no JavaScript used.

---

## 🚀 How to Run

```bash
git clone https://github.com/saimali1254/Simple_login_ui.git
cd Simple_login_ui
```

Then just open `index.html` in your browser — no setup needed.
